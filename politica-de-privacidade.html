Politica de Privacidade do App Ubezap

Este aplicativo coleta alguns dados pessoais de seus usuários.

Tipos de dados coletados

Entre os tipos de Dados Pessoais que este Aplicativo coleta, encontram-se: Posição geográfica, Cookie e Dados de Utilização. Outros Dados Pessoais coletados podem ser descritos em outras seções desta política de privacidade ou por texto de explicação dedicado contextualmente com a Coleta de Dados. Os Dados Pessoais podem ser fornecidos gratuitamente pelo usuário, ou recolhidos automaticamente ao utilizar esta Aplicativo, de acordo com o recurso utilizado. Qualquer uso de Cookies – ou de outras ferramentas de rastreamento – por este Aplicativo ou seu gestor serve para identificar Usuários e lembrar suas preferências, com o único propósito de fornecer o serviço requerido por o usuário. O não fornecimento de determinados Dados Pessoais poderá tornar impossível que este aplicativo forneça os seus serviços. O Usuário assume a responsabilidade pelos Dados Pessoais de terceiros publicados ou compartilhados através deste Aplicativo e declara ter o direito de comunicá-los ou transmiti-los, assumindo toda responsabilidade das informações fornecidas.

Modo e local de processamento dos dados

Métodos de processamento

O Data Controller processa os Dados dos Usuários de maneira adequada e deve tomar as medidas de segurança apropriadas para impedir o acesso não autorizado, divulgação, modificação ou destruição não autorizada dos Dados. O processamento de dados é realizado utilizando computadores e / ou ferramentas habilitadas por TI, seguindo procedimentos organizacionais e modos estritamente relacionados aos fins indicados. Além do Controlador de Dados, em alguns casos, os Dados podem ser acessíveis a certos tipos de pessoas responsáveis, envolvidos com a operação do site (administração, vendas, marketing, jurídico, administração do sistema).

Tempo de retenção

Os Dados são mantidos pelo tempo necessário para fornecer o serviço solicitado pelo Usuário, ou declarado pelos propósitos descritos neste documento, e o Usuário sempre pode solicitar que suspenda ou remova os dados, caso não queira mais utilizar o aplicativo.

O uso dos dados coletados

Os dados relativos ao usuário do aplicativo são recolhidos para permitir que a aplicação forneça os seus serviços, bem como para os seguintes fins: acesso a contas de serviços de terceiros, interações com base em localização, comentário de conteúdo e interação com redes e plataformas sociais externas.

Informação básica

Por padrão, isso inclui determinados Dados do Usuário, como ID, nome, imagem, gênero e sua localidade. Determinadas conexões do Usuário, como os Amigos, também estão disponíveis. Se o usuário tiver feito mais de seus dados públicos, mais informações estarão disponíveis, como: Checkin, curtidas, fotos, conteúdos públicos. Etc..

Comentário de conteúdos

Os serviços de comentários de conteúdo permitem aos Usuários fazer e publicar seus comentários sobre o conteúdo deste Aplicativo. Dependendo das configurações escolhidas pelo Proprietário, os usuários também podem deixar comentários anônimos. Se houver um endereço de e-mail entre os Dados Pessoais fornecidos pelo Usuário, ele poderá ser usado para enviar notificações de comentários sobre o mesmo conteúdo. Os usuários são responsáveis pelo conteúdo de seus próprios comentários.

Interação com redes e plataformas sociais externas

Esses serviços permitem a interação com redes sociais ou outras plataformas externas diretamente das páginas deste Aplicativo. A interação e as informações obtidas por este aplicativo estão sempre sujeitas às configurações de privacidade do usuário para cada rede social. Ex: Facebook, Instagram, Twitter, dentre outras.

Interações baseadas em localização

Este aplicativo pode coletar, usar e compartilhar dados de localização do usuário para fornecer serviços baseados em localização. A maioria dos navegadores e dispositivos fornece ferramentas para desativar esse recurso por padrão. Se for fornecida autorização explícita, os dados de localização do Utilizador poderão ser rastreados por esta Aplicação. Dados pessoais coletados: Posição geográfica.

Informações adicionais sobre Coleta e processamento de dados

Ação legal

Os Dados Pessoais do Usuário podem ser utilizados para fins legais pelo Controlador de Dados, no Tribunal ou nos estádios que conduzem a possíveis ações legais decorrentes do uso indevido deste aplicativo ou dos serviços relacionados. O Usuário está ciente do fato de o Controlador de Dados poder ser obrigado a revelar dados pessoais a pedido das autoridades públicas.

Registros e manutenção do sistema

Para fins de operação e manutenção, esta Aplicação e quaisquer serviços de terceiros podem coletar arquivos que registram interação com esta Aplicação (Logs do Sistema) ou usar outros Dados Pessoais (tais como Endereço IP).

Informações não contidas nesta política

Mais detalhes sobre a coleta ou processamento de Dados Pessoais podem ser solicitados ao Controlador de Dados a qualquer momento. Consulte as informações de contato no início deste documento.

Os direitos dos usuários

Os usuários têm o direito, a qualquer momento, de saber se os seus Dados Pessoais foram armazenados e podem consultar o Controlador de Dados para conhecer o seu conteúdo e origem, verificar a sua exatidão ou pedir que sejam completados, cancelados, atualizados ou corrigidos. As solicitações devem ser enviadas ao Controlador de Dados nas informações de contato.

Este aplicativo não oferece suporte a solicitações ” Acompanhar Pedidos”.

Aos serviços de terceiros que utiliza venda e compras, os pedidos não acompanham código de rastreio.

Alterações a esta política de privacidade

O Controlador de Dados (Administrador do Aplicativo) se reserva o direito de fazer alterações a esta política de privacidade a qualquer momento, dando aviso aos seus Usuários nesta página. É altamente recomendável verificar esta página frequentemente, referindo-se à data da última modificação listada na parte inferior. Se um Usuário se opuser a qualquer das alterações à Política, o Usuário deverá cessar de usar esta Aplicação e poderá solicitar que o Controlador de Dados apague os Dados Pessoais. Salvo indicação em contrário, a política de privacidade vigente aplica-se a todos os Dados Pessoais que o Controlador de Dados tem sobre os usuários.

Definições e referências legais

Do utilizador (USUÁRIO): O indivíduo que utiliza esta Aplicação, que deve coincidir com ou ser autorizado pelo Sujeito a Dados, a quem os Dados Pessoais se referem.

Dados Sujeitos: A pessoa singular ou coletiva a que se referem os Dados Pessoais.

Processador de Dados (ou Supervisor de Dados): A pessoa singular, pessoa coletiva, administração pública ou qualquer outro organismo, associação ou organização autorizada pelo Controlador de Dados a processar os Dados Pessoais em conformidade com esta política de privacidade.

Controlador de dados (proprietário, ou Administrador do Aplicativo): A pessoa singular, a pessoa coletiva, a administração pública ou qualquer outro organismo, associação ou organização com o direito, também em conjunto com outro responsável pelo tratamento de dados, de tomar decisões relativas aos fins e aos métodos de tratamento dos dados pessoais e dos meios utilizados, Medidas de segurança relativas ao funcionamento e à utilização do presente pedido. O Controlador de Dados, salvo indicação em contrário, é o Proprietário desta Aplicação.
